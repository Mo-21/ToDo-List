# ToDo-List

This is a To-Do List project built using JavaScript, HTML, SASS, Webpack, and Bootstrap 5. It allows you to create, edit, and delete tasks, helping you stay organized and keep track of your tasks and priorities.

## Features

1. **Create Tasks**: Easily add new tasks to your to-do list.

2. **Edit Tasks**: Double-click on a task to edit its content.

3. **Delete Tasks**: Remove tasks you no longer need.

4. **Mark as Done**: Mark tasks as completed with a single click.

5. **Responsive Design**: The application is responsive and works on various screen sizes.

6. **Data Persistence**: Your to-do list is saved in your browser's local storage, so your tasks are not lost even after you close the browser.

7. **Clean and Intuitive UI**: The project uses Bootstrap 5 for a clean and user-friendly interface.

8. **Create Folder**: the program supports the ability to add multiple folders and tasks within.
